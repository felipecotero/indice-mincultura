# Índice de Páginas Web — Ministerio de las Culturas

Directorio centralizado de todas las páginas web activas del Ministerio de las Culturas, las Artes y los Saberes.

## 📋 Contenido

Este repositorio contiene una página índice que agrupa **16 páginas web** organizadas en las siguientes categorías:

### 🎨 Artes para la Paz
- Landing page principal del programa
- Mapas interactivos de redes culturales (general, Hip Hop, Mujeres, Suroccidente)
- Plan de Movilidad Creativa — CELAC

### 📊 Infografías y Reportes
- Infografías Ley 630 de Cultura
- Infografías Cultura de Paz
- Informe Final Convenio 1022-2025
- Infografías Red Suroccidente

### 🤝 Red Intercultural Juvenil
- Cronograma Cultura es Dignidad Mayor
- Directorio Vivo RIJ

### 🎭 Cultura Viva Comunitaria
- VII Congreso Todas las Voces 2026

### 📈 Dashboards
- Dashboard Red Comunitaria de Hip Hop
